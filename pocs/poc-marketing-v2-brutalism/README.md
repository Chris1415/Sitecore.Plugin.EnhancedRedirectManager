# Redirect Manager — Marketing POC V2 — Neo Brutalism

A creative exploration that goes the other way from premium SaaS. Anti-design as design: thick black borders, hard offset shadows (no blur), no rounded corners, saturated color blocks, chunky display type (`Archivo Black`) paired with monospace body (`JetBrains Mono`), and microcopy with attitude. Status pills are rubber stamps — rotated 2-3deg, sloppy edges. Buttons "stamp" on click (the offset shadow disappears as the button translates into the page). Hover wobble, type-on hero, and `steps()`-eased number tickup keep the motion mechanical, not smooth. Four frames — `index.html`, `full-page.html`, `context-panel.html`, `dashboard-widget.html` — cross-linked via the topbar nav and a top-of-page frame switcher. Open `index.html` directly in a browser, or run `npx serve` from this folder. Honors `prefers-reduced-motion`.
