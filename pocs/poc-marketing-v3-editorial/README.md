# Redirect Manager — V3 Award Editorial

A magazine-portfolio creative exploration for the Redirect Manager Marketplace app: what if a Sitecore plugin were on Awwwards? Bold serif display (Fraunces) paired with Inter, kinetic letter-by-letter reveals, a lerp-lagged custom cursor, asymmetric editorial grid, hover word-break shifts, and slow elegant motion (cubic-bezier 0.65/0/0.35/1). One confident accent color per page — terracotta on the index and full-page, dusty olive on the panel, indigo on the widget. The three extension-point surfaces are framed as a literary volume in three parts: Part I (Full Page) is a long-form archive of redirect maps presented as journal entries with an oversized italic-serif stat pull; Part II (Panel) is a quiet sidebar marginalia with vertical-rule sections; Part III (Widget) is a dispatch tile with numbers written twice — numerals and in prose. Open `index.html` to begin.

## Run

```
npx serve .
```

## Frames

- `index.html` — landing volume, three-part table of contents
- `full-page.html` — Part I, the archive
- `context-panel.html` — Part II, the panel
- `dashboard-widget.html` — Part III, the widget
